# TicTacToe-CPP
This is an advanced Tic Tac Toe game implemented in C++ with a menu-driven interface, clean visuals, and win/draw detection.
## Features
- 2D array implementation for the board
- Menu-driven interface
- Win and Draw detection
- Nested loops and functions for better structure
- Clean board display
## How To Run
1. Open `tictactoe.cpp` in any C++ compiler (like Dev C++, Code::Blocks, or Programiz Online Compiler)
2. Compile the code
3. Run the program
4. Follow the on-screen menu to play the game
